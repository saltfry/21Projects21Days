# 21 Projects in 21 Days: Batch 4

This repository contains my solutions and project implementations for the GeeksforGeeks **21 Projects in 21 Days (Batch 4)** course. The curriculum focuses on building end-to-end applications across Machine Learning, Deep Learning, and Generative AI.

## Project Directory

### Week 1: Machine Learning & Predictive Analytics

1. **Titanic Survival Prediction:** Binary classification and data exploration.
2. **Netflix Content Strategy:** Exploratory Data Analysis (EDA) on streaming trends.
3. **Housing Market Predictor:** Regression modeling for real estate valuation.
4. **Heart Disease Predictor:** Supervised learning for healthcare diagnostics.
5. **Customer Segmentation:** Unsupervised learning using K-Means clustering.
6. **Store Sales Forecasting:** Time-series analysis for retail demand.
7. **Customer Churn Prevention:** Feature engineering and classification.

### Week 2: Deep Learning & Computer Vision

8. **Digit Recognizer:** Building a Neural Network from scratch.
9. **Image Classification:** Implementing Transfer Learning with pre-trained models.
10. **Neural Style Transfer:** Generating artistic images using CNNs.
11. **Hugging Face Pipelines:** NLP implementations for sentiment and translation.
12. **Real-Time Object Detection:** Computer vision for live video streams.
13. **Time Series Forecasting (DL):** Utilizing RNNs and LSTMs for sequential data.
14. **Custom GPT Model:** Developing a text generation engine.

### Week 3: Generative AI & Agentic AI

15. **NL to SQL Generator:** Transforming natural language into database queries.
16. **Smart OCR Bot:** Document automation and intelligent text extraction.
17. **AI Search Engine:** Building a custom search interface using LLMs.
18. **AI Browser Agent:** Automating web-based workflows.
19. **RAG Chatbot:** Retrieval-Augmented Generation using LangChain.
20. **AI Newsletter Pipeline:** End-to-end automation for content curation.

## Technical Stack

* **Language:** Python 3.x
* **Data Science:** NumPy, Pandas, Matplotlib, Seaborn
* **Machine Learning:** Scikit-learn
* **Deep Learning:** TensorFlow, PyTorch, Keras
* **GenAI & Tools:** OpenAI API, Hugging Face, LangChain, OpenCV

## Getting Started

### Prerequisites

Ensure you have Python installed. It is recommended to use a virtual environment:

```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

```

### Installation

1. Clone the repository:
```bash
git clone https://github.com/saltfry/21Projects21Days.git

```


2. Install the required dependencies:
```bash
pip install -r requirements.txt

```
